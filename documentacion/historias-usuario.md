### Historias de Usuario
* COMO usuario, QUIERO crear un tablero proporcionando mi correo electrónico PARA iniciar un espacio de trabajo colaborativo mediante acceso ligero sin registro tradicional. 
* Como propietario, QUIERO crear y gestionar múltiples listas en el tablero PARA organizar los distintos estados de mi flujo de trabajo. 
* Como propietario, QUIERO disponer de una lista especial por defecto para tarjetas completadas PARA mantener el flujo de trabajo organizado automáticamente. 
* Como propietario, QUIERO crear tarjetas de tipo "tarea" y de tipo "checklist" PARA gestionar diferentes niveles de complejidad en el trabajo. 
* Como colaborador con permisos, QUIERO marcar una tarjeta como completada y moverla a la lista correspondiente PARA visibilizar el progreso finalizado ante el equipo. 
* Como colaborador con permisos, QUIERO mover tarjetas libremente entre las distintas listas (siempre que el tablero no esté congelado) PARA reflejar el avance en tiempo real. 
* Como propietario, QUIERO definir etiquetas con colores distintivos PARA clasificar visualmente las tarjetas según su categoría. 
* COMO usuario, QUIERO filtrar las tarjetas del tablero mediante etiquetas PARA localizar rápidamente tareas específicas. 
* Como propietario, QUIERO poder congelar temporalmente el tablero PARA impedir la creación de nuevas tarjetas mientras gestiono el movimiento de las actuales.
* Como propietario, QUIERO establecer un límite máximo (N) de tarjetas por lista PARA controlar la carga de trabajo y evitar cuellos de botella. 
* Como propietario, QUIERO configurar reglas de flujo que obliguen a las tarjetas a para por determinadas listas PARA asegurar el cumplimiento de los procesos internos. 
* COMO usuario, QUIERO que el sistema mantenga un historial de trazabilidad completo de todas las acciones PARA conocer quién y cuándo realizó cada cambio en el tablero.
